JavaScript Storage Types

A comprehensive guide to browser storage mechanisms

🍪

Cookies

Small pieces of data stored in the user's browser, sent with every HTTP request. Limited to about 4KB.

💾

Local Storage

Persistent key-value storage that remains until explicitly cleared. Typically 5-10MB per domain.

🔄

Session Storage

Temporary key-value storage that clears when the browser tab closes. Typically 5-10MB per domain.

🗄️

IndexedDB

A full-featured NoSQL database for storing large amounts of structured data. Can store hundreds of MBs.

Storage Comparison

Storage Type Capacity Persistence Accessible From Sent to Server Use Cases
Cookies ~4KB Expires based on set expiry Any window Yes, with every request Session management, personalization
Local Storage 5-10MB Until explicitly cleared Same origin No Persistent app data, user preferences
Session Storage 5-10MB Tab/window session Same tab No Form data, temporary state
IndexedDB Up to 50% of disk space Until explicitly cleared Same origin No Large datasets, offline apps
